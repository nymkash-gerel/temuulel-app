## 1. Эхлэх

### 1.1 Бүртгэл үүсгэх

Temuulel платформд бүртгүүлэхдээ дараах алхмуудыг дагана уу:

1. **app.temuulel.mn** хаягаар орно.
2. **"Бүртгүүлэх"** товчийг дарна.
3. И-мэйл хаяг болон нууц үгээ оруулна.
4. Баталгаажуулах и-мэйл ирэх тул и-мэйл хайрцагаа шалгана.
5. И-мэйл дэх холбоосоор баталгаажуулна.

Бүртгэл амжилттай үүссэний дараа хянах самбар (dashboard) руу шууд нэвтэрнэ.

### 1.2 Дэлгүүр тохируулах

Эхний удаа нэвтрэхэд дэлгүүрийн мэдээллээ оруулна:

1. **Тохиргоо > Дэлгүүрийн мэдээлэл** хэсэгт орно.
2. Дараах мэдээллийг бөглөнө:
   - **Дэлгүүрийн нэр** — Таны бизнесийн нэр (жишээ: "Ногоон дэлгүүр")
   - **Лого** — Дэлгүүрийн лого зураг оруулах
   - **Холбоо барих утас** — Харилцагчдад харагдах утасны дугаар
   - **Хаяг** — Дэлгүүрийн байршил
   - **Тайлбар** — Дэлгүүрийн товч танилцуулга
3. **"Хадгалах"** товчийг дарна.

### 1.3 Загвар сонгох

Temuulel нь бизнесийн төрлөөс хамааран бэлэн загвар санал болгодог. Загвар нь чатбот урсгал, бүтээгдэхүүний ангилал, хүргэлтийн тохиргоог автоматаар тохируулна.

Боломжит загварууд:

| Загвар | Тайлбар | Тохирох бизнес |
|--------|---------|----------------|
| **Жижиглэн худалдаа** | Хувцас, гутал, аксессуар борлуулах | Онлайн дэлгүүр, бутик |
| **Ресторан** | Хоол захиалга, ширээ захиалга | Ресторан, кафе, хоолны газар |
| **Кэмпинг** | Гэр, майхан, талбай захиалга | Жуулчны бааз, кэмпинг |
| **Үйлчилгээ** | Цаг захиалга, үйлчилгээ | Гоо сайхан, засвар үйлчилгээ |

Загвар сонгохдоо:

1. **Тохиргоо > Чат урсгал** хэсэгт орно.
2. **"Загвараас үүсгэх"** товчийг дарна.
3. Бизнесийн төрлөө сонгоно.
4. Бэлэн урсгал автоматаар үүснэ. Дараа нь өөрийн хэрэгцээнд тохируулж засварлах боломжтой.

### 1.4 Төлбөрийн план

Temuulel дөрвөн план санал болгоно:

| План | Үнэ | Онцлог |
|------|-----|--------|
| **Free** | Үнэгүй | Суурь функц, хязгаарлагдмал мессеж |
| **Basic** | 29,900₮/сар | Илүү олон бүтээгдэхүүн, мессеж |
| **Pro** | 79,900₮/сар | Бүрэн функц, AI зөвлөмж |
| **Enterprise** | Тохиролцоно | Хязгааргүй, тусгай дэмжлэг |

Планаа сонгохдоо **Тохиргоо > Төлбөрийн план** хэсэгт орно.

## 2. Бүтээгдэхүүн

### 2.1 Бүтээгдэхүүн нэмэх

Шинэ бүтээгдэхүүн нэмэхдээ:

1. Зүүн цэснээс **"Бүтээгдэхүүн"** хэсэгт орно.
2. Баруун дээд буланд байрлах **"Нэмэх"** товчийг дарна.
3. Дараах мэдээллийг бөглөнө:

**Үндсэн мэдээлэл:**
- **Нэр** — Бүтээгдэхүүний нэр (заавал)
- **Тайлбар** — Дэлгэрэнгүй тайлбар
- **Ангилал** — Хувцас, Гутал, Цүнх, Гоёл чимэглэл
- **Дэд ангилал** — Ангилал сонгосны дараа сонголт гарна (жишээ: Цамц, Өмд, Даашинз)
- **SKU код** — Бүтээгдэхүүний дотоод код
- **Суурь үнэ** — Үндсэн үнэ (төгрөгөөр)

**Нэмэлт мэдээлэл:**
- **Хэмжээний зөвлөмж** (Fit note) — Чатбот хэмжээний талаар асуухад ашиглана
- **AI контекст** — AI-д нэмэлт мэдээлэл өгөх (жишээ: "Энэ бүтээгдэхүүн захиалгаар ирдэг, 7-14 хоног хүлээнэ")
- **Facebook Post ID** — Facebook нийтлэлийн ID
- **Instagram Post ID** — Instagram нийтлэлийн ID

**Төлөв:**
- **Ноорог** — Зөвхөн та харна, чатбот санал болгохгүй
- **Идэвхтэй** — Чатбот хэрэглэгчдэд санал болгоно

4. **"Хадгалах"** товчийг дарна.

### 2.2 Хувилбар (Variants)

Нэг бүтээгдэхүүнд олон хувилбар (хэмжээ, өнгө) нэмэх боломжтой.

1. Бүтээгдэхүүн нэмэх/засах хуудсанд **"Хувилбартай"** товчийг идэвхжүүлнэ.
2. Хувилбар бүрт дараах мэдээллийг оруулна:
   - **Хэмжээ** — S, M, L, XL гэх мэт
   - **Өнгө** — Хар, Цагаан, Улаан гэх мэт
   - **Үнэ** — Хувилбарын үнэ (суурь үнээс өөр бол)
   - **Нөөц** — Тухайн хувилбарын нөөцийн тоо
   - **SKU** — Хувилбарын код

3. **"+ Хувилбар нэмэх"** товчоор шинэ хувилбар нэмнэ.
4. Хувилбар устгахдаа хажууд байрлах **"Устгах"** товчийг дарна.

Нөөц дуусахад ойрхон (тохируулсан босго хэмжээнд хүрэхэд) систем автоматаар **"Нөөц дуусаж байна"** мэдэгдэл илгээнэ.

### 2.3 Зураг оруулах

Бүтээгдэхүүнд зураг нэмэхдээ:

1. Бүтээгдэхүүний засварлах хуудсанд **"Зураг нэмэх"** хэсгийг олно.
2. Файл сонгох эсвэл зургийг чирж тавина.
3. Олон зураг нэмэх боломжтой. Эхний зураг нь үндсэн зураг болно.
4. Зургийн дарааллыг чирж өөрчлөх боломжтой.

Зургууд Supabase Storage-д хадгалагдах бөгөөд чатбот болон дэлгүүрийн хуудсанд автоматаар харагдана.

### 2.4 Категори (Ангилал)

Бүтээгдэхүүний ангилал нь AI чатботод хайлтыг хялбарчилна. Үндсэн ангилалууд:

- **Хувцас** — Цамц, Өмд, Даашинз, Пальто, Куртка
- **Гутал** — Пүмп, Туфли, Кроссовок
- **Цүнх** — Гар цүнх, Нуруун цүнх, Бэлтгэн цүнх
- **Гоёл чимэглэл** — Бүс, Малгай, Ороолт, Бээлий

Та **Тохиргоо > Бүтээгдэхүүний тохиргоо** хэсгээс шинэ ангилал нэмж, засварлах боломжтой.

### 2.5 Excel/CSV импорт

Олон бүтээгдэхүүнийг нэг дор оруулахдаа:

1. **Бүтээгдэхүүн > Олноор оруулах** хэсэгт орно.
2. Excel (.xlsx, .xls) эсвэл CSV файлаа сонгоно.

**Файлын формат:**

Дараах баганууд байх ёстой:

| Багана | Монгол нэр | Тайлбар |
|--------|-----------|---------|
| `name` | `нэр` | Бүтээгдэхүүний нэр (заавал) |
| `price` | `үнэ` | Үнэ (заавал) |
| `stock` | `нөөц` / `тоо` | Нөөцийн тоо |
| `category` | `ангилал` | Ангилал |
| `description` | `тайлбар` | Тайлбар |
| `size` | `хэмжээ` | Хэмжээ |
| `color` | `өнгө` | Өнгө |
| `image` | `зураг` | Зургийн URL |
| `facebook_post_id` | — | Facebook нийтлэлийн ID |
| `instagram_post_id` | — | Instagram нийтлэлийн ID |

Баганын нэрийг монгол эсвэл англиар бичиж болно. Систем хоёуланг нь ойлгоно.

**Жишээ CSV:**

```csv
name,description,category,price,stock,size,color
Эмэгтэй цамц,Зуны цамц,clothing,25000,10,M,Цагаан
Эрэгтэй өмд,Хөнгөн өмд,clothing,45000,5,L,Хар
Гар цүнх,Арьсан цүнх,bags,89000,3,,Бор
```

3. Файл сонгосны дараа **"Урьдчилан харах"** хүснэгтэд бүтээгдэхүүнүүд харагдана.
4. **"Оруулах"** товчийг дарна.
5. Амжилттай импортын дараа бүтээгдэхүүн жагсаалт руу шилжинэ.

**Жишээ файл татах** товчоор формат заавар бүхий жишээ файл татах боломжтой.

### 2.6 AI баяжуулалт (AI Enrichment)

Temuulel-ийн AI баяжуулалт нь бүтээгдэхүүний мэдээллийг автоматаар сайжруулна:

- Excel-ээс оруулсан бүтээгдэхүүнүүдэд автоматаар AI баяжуулалт хийгдэнэ
- AI нь бүтээгдэхүүний нэр, тайлбараас ангилал, түлхүүр үг, чатботын хариултыг оновчилно
- Гараар ажиллуулахдаа бүтээгдэхүүний засварлах хуудсанд **"AI баяжуулах"** товчийг дарна

AI баяжуулалт дараах зүйлсийг хийнэ:

1. **Тайлбар сайжруулах** — Товч мэдээллээс дэлгэрэнгүй тайлбар бичнэ
2. **Ангилал санал болгох** — Зөв ангилалд оруулна
3. **Түлхүүр үг** — Чатбот хайлтад ашиглах түлхүүр үг нэмнэ
4. **Хэмжээний зөвлөмж** — Бүтээгдэхүүний онцлогоос хэмжээний зөвлөмж гаргана

## 3. Захиалга

### 3.1 Захиалга харах

Захиалгын жагсаалтыг харахдаа:

1. Зүүн цэснээс **"Захиалга"** хэсэгт орно.
2. Дэлгэцийн дээд хэсэгт тоон мэдээлэл харагдана:
   - **Хүлээгдэж буй** — Шинэ захиалга
   - **Боловсруулж буй** — Баталгаажсан + Бэлтгэж буй
   - **Илгээсэн** — Хүргэлтэнд гарсан
   - **Хүргэсэн** — Амжилттай хүргэсэн

**Шүүлтүүр ашиглах:**

- **Хайлт** — Захиалгын дугаар, харилцагчийн нэр, утасны дугаараар хайна
- **Төлөв** — Хүлээгдэж буй, Баталгаажсан, Бэлтгэж буй, Илгээсэн, Хүргэсэн, Цуцлагдсан
- **Хугацаа** — Өнөөдөр, Энэ долоо хоног, Энэ сар

**Экспорт:**

Захиалгыг Excel эсвэл CSV форматаар татаж авахдаа баруун дээд буланд байрлах **"Excel"** эсвэл **"CSV"** товчийг дарна.

### 3.2 Захиалгын дэлгэрэнгүй

Захиалгын мөрийг дарж дэлгэрэнгүй харна:

- **Харилцагчийн мэдээлэл** — Нэр, утас, хаяг
- **Бүтээгдэхүүнүүд** — Захиалсан бүтээгдэхүүн, тоо, үнэ
- **Төлбөрийн мэдээлэл** — Төлбөрийн арга, төлөв
- **Хүргэлтийн мэдээлэл** — Хаяг, хүргэлтийн төлбөр
- **Төлвийн түүх** — Захиалгын төлөв хэрхэн өөрчлөгдсөнийг харна

### 3.3 Төлөв шинэчлэх

Захиалгын төлвийг дараах дарааллаар шинэчилнэ:

```
Хүлээгдэж буй (pending)
    |
    v
Баталгаажсан (confirmed)
    |
    v
Бэлтгэж буй (processing)
    |
    v
Илгээсэн (shipped)
    |
    v
Хүргэсэн (delivered)
```

Эсвэл аль ч шатанд **Цуцлагдсан (cancelled)** төлөв рүү шилжүүлж болно.

**Төлөв шинэчлэхдээ:**

1. Захиалгын дэлгэрэнгүй хуудсанд орно.
2. Төлөвийн товчуудаас дараагийн төлвийг сонгоно.
3. Тэмдэглэл нэмж болно (заавал биш).
4. **"Хадгалах"** товчийг дарна.

Төлөв шинэчлэхэд дараах зүйлс автоматаар болно:
- Дэлгүүрийн эзэнд мэдэгдэл илгээнэ (и-мэйл + апп дотор + push)
- Webhook тохируулсан бол гадны системд дамжуулна
- Нөөц шинэчлэгдэнэ

### 3.4 Төлбөр

Temuulel гурван төлбөрийн аргыг дэмждэг:

#### QPay

QPay нь Монголын хамгийн түгээмэл цахим төлбөрийн систем.

1. Харилцагч захиалга өгнө.
2. Систем QPay QR код автоматаар үүсгэнэ.
3. Харилцагч банкны аппликейшнээрээ QR код уншуулна.
4. Төлбөр амжилттай болсон тухай QPay системээс Temuulel руу мэдэгдэл ирнэ.
5. Захиалгын төлбөрийн төлөв **"Төлөгдсөн"** болно.

**QPay тохируулахдаа:** Тохиргоо > Төлбөр хүлээн авах > QPay хэсэгт мерчант мэдээллээ оруулна.

#### Банкны шилжүүлэг

1. Харилцагч банкны дансаар шилжүүлэг хийнэ.
2. Дэлгүүрийн эзэн төлбөр ирсэн эсэхийг шалгаж, захиалгыг гараар баталгаажуулна.

#### Бэлнээр (Хүргэлтийн үед)

1. Харилцагч хүргэлтийн үед бэлнээр төлнө.
2. Жолооч төлбөр хүлээж авсны дараа захиалгыг баталгаажуулна.

### 3.5 Буцаалт (Returns & Refunds)

Буцаалтын хүсэлт удирдахдаа:

1. Зүүн цэснээс **"Буцаалт"** хэсэгт орно.
2. Дэлгэцийн дээд хэсэгт статистик харагдана:
   - **Хүлээгдэж буй** — Шийдвэрлэгдээгүй буцаалт
   - **Зөвшөөрсөн** — Зөвшөөрөгдсөн буцаалт
   - **Дууссан** — Буцаалт хийгдсэн
   - **Татгалзсан** — Татгалзсан буцаалт

**Буцаалт шийдвэрлэхдээ:**

1. Буцаалтын мөрийг дарж дэлгэрэнгүй хуудсанд орно.
2. Буцаалтын мэдээллийг хянана:
   - **Төрөл** — Бүтэн буцаалт эсвэл хэсэгчилсэн
   - **Шалтгаан** — Харилцагчийн бичсэн шалтгаан
   - **Буцаах дүн** — Буцаах мөнгөн дүн
3. **"Зөвшөөрөх"** эсвэл **"Татгалзах"** товчийг дарна.
4. Зөвшөөрсөн тохиолдолд буцаалтын мөнгийг шилжүүлж, **"Дууссан"** төлөв рүү шилжүүлнэ.

**Буцаалтын бодлого:**

Та **Тохиргоо > Chatbot тохиргоо > Буцаалт/Солилтын бодлого** хэсэгт бодлогоо бичнэ. Жишээ:

> "Бараа хүлээн авснаас хойш 14 хоногийн дотор буцаах боломжтой. Шошго хадгалагдсан, хэрэглээгүй байх шаардлагатай. Буцаалтын хураамж 5,000₮."

AI чатбот энэ бодлогоор хэрэглэгчдэд автоматаар хариулна. Бодлого бичээгүй бол чатбот "Менежерээс лавлана уу" гэж хариулна.

## 4. Хүргэлт

### 4.1 Хүргэлт үүсгэх

Шинэ хүргэлт үүсгэхдээ:

1. Зүүн цэснээс **"Хүргэлт"** хэсэгт орно.
2. **"+ Хүргэлт нэмэх"** товчийг дарна.
3. Дараах мэдээллийг бөглөнө:
   - **Хүргэх хаяг** — Хүлээн авагчийн хаяг (заавал)
   - **Хүлээн авагчийн нэр** — Нэр
   - **Утас** — Холбоо барих утас
   - **Хүргэлтийн төрөл** — Өөрийн жолооч эсвэл Гадны хүргэлт
   - **Хүргэлтийн төлбөр** — Хүргэлтийн үнэ
   - **Жолооч** — Жолооч оноох (сонголтоор)
   - **Тэмдэглэл** — Нэмэлт мэдээлэл
4. **"Хүргэлт үүсгэх"** товчийг дарна.

Захиалгыг **"Илгээсэн"** төлөв рүү шилжүүлэхэд хүргэлт автоматаар үүсэх тохиргоо хийх боломжтой.

### 4.2 Жолооч оноох

Хүргэлтэнд жолооч оноох хоёр арга бий:

#### Гараар оноох

1. Хүргэлтийн жагсаалтаас хүргэлтийг сонгоно.
2. Жолоочийн цэснээс жолоочоо сонгоно.
3. Хадгална.

#### AI-аар оноох

1. Хүргэлтийн мөрийн **"AI оноох"** товчийг дарна.
2. AI дараах шалгууруудыг ашиглан жолооч санал болгоно:
   - **Хамгийн ойрхон жолооч** — Байршил ойр жолоочийг сонгоно
   - **Хамгийн бага ачаалалтай** — Идэвхтэй хүргэлт цөөнтэй жолоочийг сонгоно
   - **Тээврийн хэрэгсэл тохирох** — Захиалгын хэмжээнд тохирох тээвэртэй жолоочийг сонгоно
   - **Ээлжлэн хуваарилах** — Жолоочид жигд хуваарилна
   - **Амжилтын хувиар** — Хамгийн амжилттай жолоочийг сонгоно

**AI оноолтын горим тохируулах:**

Тохиргоо > Хүргэлтийн тохиргоо хэсэгт гурван горим байна:
- **Автомат** — AI жолоочийг автоматаар оноох
- **Санал болгох** — AI санал болгоно, та зөвшөөрнө
- **Гараар** — Та өөрөө жолооч оноох

### 4.3 Хүргэлтийн бүс, төлбөр

Хүргэлтийн бүс болон төлбөрийг тохируулахдаа:

1. **Тохиргоо > Хүргэлт** хэсэгт орно.
2. Хүргэлтийн бүс тус бүрт тохируулна:
   - **Бүсийн нэр** — Жишээ: Улаанбаатар хотын төв, Хан-Уул дүүрэг
   - **Хүргэлтийн төлбөр** — Тухайн бүсийн хүргэлтийн үнэ
   - **Үнэгүй хүргэлтийн босго** — Энэ дүнгээс дээш захиалгад хүргэлт үнэгүй

Захиалга үүсэхэд систем хүргэлтийн хаягаар бүсийг тодорхойлж, хүргэлтийн төлбөрийг автоматаар тооцоолно.

### 4.4 Цагийн хуваарь

Хүргэлтийн ажлын цагийг тохируулахдаа:

1. **Тохиргоо > Хүргэлтийн тохиргоо** хэсэгт орно.
2. **Ажлын цаг** хэсэгт эхлэх болон дуусах цагийг тохируулна (жишээ: 09:00 - 22:00).
3. Ажлын бус цагт хүргэлт оноогдохгүй.

### 4.5 Хүргэлтийн төлөв мөшгих (Tracking)

**Дэлгүүрийн эзэн:**

Хүргэлтийн жагсаалтаас бүх хүргэлтийн төлвийг хянах боломжтой:

| Төлөв | Тайлбар |
|-------|---------|
| Хүлээгдэж буй | Хүргэлт үүссэн, жолооч оноогоогүй |
| Оноосон | Жолоочид оноосон |
| Авсан | Жолооч бараа авсан |
| Зам дээр | Хүргэлтэнд явж буй |
| Хүргэсэн | Амжилттай хүргэсэн |
| Хоцорсон | Хүргэлт хоцорсон |
| Амжилтгүй | Хүргэлт амжилтгүй болсон |
| Цуцлагдсан | Хүргэлт цуцлагдсан |

**Газрын зураг:**

**Хүргэлт > Газрын зураг** хэсэгт бүх идэвхтэй хүргэлтийн байршлыг газрын зураг дээр харна.

**Харилцагч:**

Харилцагч хүргэлтийн мэдээллийг `track/[дугаар]` хаягаар шалгах боломжтой. Энэ холбоосыг мессежээр илгээнэ.

## 5. Жолоочийн портал

Жолоочийн портал нь хүргэлтийн жолоочдод зориулсан тусдаа интерфэйс юм.

### 5.1 Жолооч бүртгэх

**Дэлгүүрийн эзэн талаас:**

1. **"Жолоочууд"** хэсэгт орно.
2. **"+ Жолооч нэмэх"** товчийг дарна.
3. Мэдээллийг бөглөнө:
   - **Нэр** — Жолоочийн нэр (заавал)
   - **Утас** — Утасны дугаар (заавал)
   - **И-мэйл** — И-мэйл хаяг
   - **Тээврийн хэрэгсэл** — Мотоцикл, Машин, Дугуй, Явган
   - **Улсын дугаар** — Тээврийн хэрэгслийн дугаар
4. **"Нэмэх"** товчийг дарна.
5. Жолоочийн **"Бүртгэлийн линк хуулах"** товчийг дарж линкийг жолоочид илгээнэ.

**Жолооч талаас:**

1. Дэлгүүрийн эзнээс ирсэн бүртгэлийн линкээр орно.
2. И-мэйл хаяг, нууц үг бүртгүүлнэ.
3. Бүртгэл амжилттай болсны дараа жолоочийн порталд нэвтэрнэ.

### 5.2 Жолоочийн нэвтрэх

1. `/driver/login` хаягаар жолоочийн нэвтрэх хуудсанд орно.
2. Бүртгүүлсэн и-мэйл, нууц үгээ оруулна.
3. Жолоочийн хянах самбар руу шилжинэ.

### 5.3 Хүргэлт хүлээн авах

Жолоочийн хянах самбарт идэвхтэй хүргэлтүүд жагсаалтаар харагдана.

Хүргэлт бүрт дараах мэдээлэл байна:
- Хүргэлтийн дугаар
- Хүлээн авагчийн нэр, утас
- Хүргэх хаяг
- Хүргэлтийн төлбөр
- Товч үйлдлийн товч

**Төлвийг шилжүүлэх:**

Жолооч дараах дарааллаар төлвийг шинэчилнэ:

```
Оноосон  →  Авсан  →  Зам дээр  →  Хүргэсэн
                                  →  Хоцорсон
                                  →  Амжилтгүй
```

Хүргэлтийн мөрийн товч үйлдлийн товчийг дарж төлвийг нэг дарж шинэчилнэ.

### 5.4 Хүргэлтийн дэлгэрэнгүй

Хүргэлтийн мөрийг дарж дэлгэрэнгүй хуудсанд орно:

- Хүлээн авагчийн мэдээлэл, утас
- Хүргэх хаяг
- Захиалгын мэдээлэл (холбоотой захиалга байгаа бол)
- Тэмдэглэл
- Төлвийн түүх (хэзээ, хэн шинэчилсэн)

### 5.5 Маршрут оновчлох

Олон хүргэлттэй үед маршрутыг оновчлохдоо:

1. Жолоочийн хянах самбарт **"Маршрут оновчлох"** товчийг дарна.
2. AI хүргэлтийн хаягуудыг зам, зайг тооцоолж оновчтой дарааллаар эрэмбэлнэ.
3. Оновчилсон маршрутаар хүргэлтийн жагсаалт шинэчлэгдэнэ.

### 5.6 Нотлох зураг (Proof photos)

Хүргэлтийг **"Хүргэсэн"** төлөв рүү шилжүүлэхдээ нотлох зураг оруулж болно:

1. Хүргэлтийн дэлгэрэнгүй хуудсанд орно.
2. **"Зураг оруулах"** товчийг дарна.
3. Хүргэсэн газрын зургийг сонгоно.
4. Зураг хадгалагдаж, дэлгүүрийн эзэнд харагдана.

### 5.7 Орлого (Earnings)

Жолоочийн орлогыг хянахдаа:

1. **"Орлого"** хэсэгт орно.
2. Дараах мэдээлэл харагдана:
   - **Өнөөдөр** — Өнөөдрийн орлого, хүргэлтийн тоо
   - **Энэ долоо хоног** — Долоо хоногийн орлого
   - **Энэ сар** — Сарын орлого
   - **Нийт** — Нийт орлого
3. Доод хэсэгт хүргэлтийн түүх жагсаалтаар харагдана.

**Жолоочийн цалин:**

Дэлгүүрийн эзэн **"Жолоочийн цалин"** хуудаснаас цалин тооцоолж, хүргэлтийн тоо болон хүргэлтийн төлбөрөөс цалин бодно. Цалингийн төлөв: Хүлээгдэж буй → Зөвшөөрсөн → Төлсөн.

### 5.8 Жолоочийн чат

Жолооч болон дэлгүүрийн эзэн хооронд шууд чатлах боломжтой:

- Жолооч: `/driver/chat` хуудаснаас мессеж илгээнэ
- Дэлгүүрийн эзэн: **"Жолоочийн чат"** хуудаснаас хариулна

## 6. Чатбот & Харилцаа

### 6.1 AI чатбот тохируулах

AI чатботыг тохируулахдаа **Тохиргоо > Chatbot тохиргоо** хэсэгт орно.

#### AI автомат хариулт

- **Идэвхтэй** болговол ирсэн бүх мессежид AI автоматаар хариулна
- **Идэвхгүй** болговол зөвхөн та гараар хариулна

#### Хариултын өнгө аяс

Гурван сонголтоос сонгоно:

| Өнгө аяс | Тайлбар |
|-----------|---------|
| **Найрсаг** | Дотно, илэн далангүй хариулт |
| **Мэргэжлийн** | Албан ёсны, товч хариулт |
| **Энгийн** | Хялбар, шууд хариулт |

#### Хэл

- **Монгол** — Зөвхөн монголоор хариулна
- **English** — Зөвхөн англиар хариулна
- **Хоёулаа** — Хэрэглэгчийн хэлээр хариулна

#### Welcome мессеж

Шинэ харилцагч анх мессеж бичихэд илгээгдэх мэндчилгээ мессеж. Жишээ:

> "Сайн байна уу! Манай дэлгүүрт тавтай морил. Танд юугаар туслах вэ?"

#### Away мессеж

Ажлын бус цагт автоматаар илгээгдэх мессеж. Жишээ:

> "Одоогоор ажлын бус цагт байна. Таны мессежийг хүлээн авсан бөгөөд удахгүй хариулах болно."

#### Түргэн хариултууд

Чат хуудаснаас нэг дарж илгээх боломжтой бэлэн хариултууд. Жишээ:
- "Баярлалаа, захиалга баталгаажлаа."
- "Хүргэлт 1-3 хоногт хийгдэнэ."
- "Та утасны дугаараа үлдээнэ үү?"

Шинэ хариулт нэмэхдээ текстээ бичээд **"+ Нэмэх"** товчийг дарна.

#### Бүтээгдэхүүн харуулалт

- **Үнэ харуулах** — Чатбот бүтээгдэхүүний үнийг хариултад оруулна
- **Хамгийн ихдээ харуулах бүтээгдэхүүн** — 3, 5, эсвэл 10 (илэрцийн тоо)

#### Виджетийн өнгө

Чат виджетийн өнгийг сонгоно: Цэнхэр, Циан, Ногоон, Нил, Шар, Улаан, Ягаан.

Баруун талд **"Урьдчилан харах"** хэсэгт виджетийн харагдах байдлыг хянах боломжтой.

### 6.2 Чат урсгал зохион байгуулах (Flow Builder)

Чат урсгал нь AI чатботоос илүү нарийвчилсан яриа зохион байгуулах боломж олгоно.

1. **Тохиргоо > Чат урсгал** хэсэгт орно.
2. **"Шинэ урсгал"** товчийг дарж шинэ урсгал үүсгэнэ, эсвэл **"Загвараас үүсгэх"** товчоор бэлэн загвар ашиглана.

**Урсгалын элементүүд:**

- **Trigger (Идэвхжүүлэгч)** — Урсгал эхлэх нөхцөл:
  - Түлхүүр үг — Тодорхой үг ашиглахад (жишээ: "захиалга")
  - Шинэ яриа — Шинэ харилцагч анх мессеж бичихэд
  - Товч дарах — Quick reply товч дарахад
  - Зорилго — AI тодорхой зорилго илрүүлэхэд
- **Мессеж** — Текст мессеж илгээх
- **Товч** — Сонголтын товчууд
- **Нөхцөл** — Хариултаас хамааран салаалах
- **Хүний оператор руу шилжүүлэх** — Ажилтанд дамжуулах

### 6.3 Messenger, Instagram, Telegram холболт

#### Facebook Messenger холбох

1. **Тохиргоо > Холболт** хэсэгт орно.
2. **"Facebook Messenger"** хэсгийн **"Холбох"** товчийг дарна.
3. Facebook-руу нэвтрэх цонх нээгдэнэ.
4. Facebook-д нэвтрэн, Page сонгоно.
5. Шаардлагатай зөвшөөрлүүдийг өгнө.

**Шаардлага:**
- Facebook Page-д Admin эрхтэй байх
- Page Messaging идэвхтэй байх

Холбогдсоны дараа:
- Messenger мессежид AI автоматаар хариулна
- Бүтээгдэхүүний карт, түргэн хариулт илгээнэ
- Шинэ харилцагч бүртгэгдэнэ
- Мессежүүд чат хуудсанд харагдана

#### Instagram DM холбох

1. **Тохиргоо > Холболт** хэсэгт орно.
2. **"Instagram DM"** хэсгийн **"Холбох"** товчийг дарна.
3. Facebook-руу нэвтрэн, Instagram Business аккаунт сонгоно.

**Шаардлага:**
- Facebook Page-д Instagram Business аккаунт холбогдсон байх
- Instagram Professional Account байх

#### Telegram холбох

Telegram bot-тоо холбох тохиргоо удахгүй нэмэгдэнэ.

#### Суваг салгах

Аливаа сувгийг салгахдаа:
1. **Тохиргоо > Холболт** хэсэгт орно.
2. Холбогдсон сувгийн **"Салгах"** товчийг дарна.
3. Баталгаажуулна.

### 6.4 Вэб виджет (Web Widget)

Та өөрийн вэбсайтад чат виджет суулгах боломжтой.

1. **Тохиргоо > Холболт** хэсэгт орно.
2. **"Вэб чат виджет"** хэсгийн кодыг хуулна.
3. Вэбсайтынхаа HTML файлын `<body>` таг дотор буулгана.

**Суулгах код:**

```html
<script src="https://app.temuulel.mn/widget.js" data-store-id="ТАНЫ_STORE_ID"></script>
```

Виджет нь:
- Вэбсайтын баруун доод буланд харагдана
- AI автомат хариулт өгнө
- Бүтээгдэхүүн хайх, захиалга өгөх боломжтой
- Дэлгүүрийн өнгө, стильд тохирно

### 6.5 Хүний оператор руу шилжүүлэх (Escalation)

AI чатбот хэрэглэгчийн асуудлыг шийдэж чадахгүй тохиолдолд хүний оператор руу автоматаар шилжүүлнэ.

#### Автомат шилжүүлэг

Дараах тохиолдолд автоматаар шилжүүлнэ:

- **Түлхүүр үг** — Хэрэглэгч "менежер", "хүн", "оператор", "гомдол" гэх мэт үг ашиглахад
- **Ухаалаг шилжүүлэг (Smart Escalation)** — AI дараах дохиог илрүүлж оноо тооцоолно:
  - Гомдол, сэтгэл дундуур байдал
  - Буцаалт, солилт хүсэлт
  - Төлбөрийн маргаан
  - Давтагдсан мессеж (нэг зүйлийг олон удаа асуух)
  - AI хариулж чадахгүй байдал
  - Удаан шийдвэрлэгдээгүй яриа

Оноо тодорхой босго (анхны тохиргоогоор 60) давахад автоматаар шилжүүлнэ.

#### Ач холбогдлын түвшин

Шилжүүлсэн ярианууд ач холбогдлын түвшинтэй байна:

| Түвшин | Тайлбар |
|--------|---------|
| **Маш чухал (Critical)** | Нэн даруй хариулах шаардлагатай |
| **Чухал (High)** | Түргэн хариулах шаардлагатай |
| **Дундаж (Medium)** | Энгийн шилжүүлэг |

#### Хүлээж авах

Чат хуудсанд шилжүүлсэн ярианууд тусгай **"Шилжсэн"** табд харагдана. Хүлээж авахдаа **"Хүлээж авах"** товчийг дарна. Үүний дараа та гараар хариулж эхэлнэ.

### 6.6 Comment Auto-Reply

Facebook, Instagram нийтлэлийн сэтгэгдэлд автоматаар хариулах:

1. **Тохиргоо > Comment Auto-Reply** хэсэгт орно.
2. Дүрмүүд тохируулна:
   - Тодорхой түлхүүр үгэнд хариулах
   - Автомат DM илгээх
   - Бүтээгдэхүүний мэдээлэл илгээх

## 7. Үйлчилгээ & Цаг захиалга

Үйлчилгээний бизнесүүдэд (гоо сайхан, ресторан, жуулчны бааз) зориулсан функц.

### 7.1 Үйлчилгээ нэмэх

1. Зүүн цэснээс **"Үйлчилгээ"** хэсэгт орно.
2. **"+ Үйлчилгээ нэмэх"** товчийг дарна.
3. Дараах мэдээллийг бөглөнө:
   - **Нэр** — Үйлчилгээний нэр (жишээ: "Маникюр")
   - **Тайлбар** — Дэлгэрэнгүй тайлбар
   - **Ангилал** — Үйлчилгээний төрөл
   - **Үргэлжлэх хугацаа** — Минутаар (жишээ: 60 мин)
   - **Үнэ** — Төгрөгөөр
   - **Зураг** — Үйлчилгээний зураг
4. **Төлөв** сонгоно:
   - **Идэвхтэй** — Хэрэглэгчид захиалах боломжтой
   - **Ноорог** — Харагдахгүй
   - **Архивлагдсан** — Хадгалагдсан боловч идэвхгүй
5. **"Хадгалах"** товчийг дарна.

### 7.2 Нөөц удирдах (Bookable Resources)

Нөөц нь физик орон зай, тоног төхөөрөмж зэргийг удирдах зориулалттай.

1. **"Нөөц / Ширээ"** хэсэгт орно.
2. **"+ Нэмэх"** товчийг дарна.

**Нөөцийн төрлүүд:**

| Төрөл | Тайлбар | Тохирох бизнес |
|-------|---------|---------------|
| **Ширээ** | Ресторан, кафены ширээ | Хоолны газар |
| **Өрөө** | Зочид буудлын өрөө | Зочид буудал |
| **Майхны талбай** | Майхан тавих талбай | Кэмпинг |
| **RV талбай** | Автомашины зогсоол | Кэмпинг |
| **Монгол гэр** | Уламжлалт гэр | Жуулчны бааз |
| **Модон байшин** | Cabin | Жуулчны бааз |

**Нөөцийн мэдээлэл:**
- **Нэр** — Жишээ: "Ширээ 4", "VIP өрөө", "Монгол гэр 1"
- **Тайлбар** — "Цонхны хажуу", "Голын эрэг" гэх мэт
- **Багтаамж** — Хүний тоо
- **Үнэ** — Нэг нэгж үнэ (₮)
- **Онцлог** — Төрлөөс хамааран:
  - Ширээ: Цонхны талтай, VIP, Гадна, Хувийн өрөө
  - Өрөө: WiFi, ТВ, Мини бар, Тагт, Угаалгын өрөө
  - Гэр: Халаалт, WiFi, Угаалгын өрөө, Уламжлалт
  - Майхны талбай: Цахилгаан, Ус, Галын талбай, Сүүдэр
  - Модон байшин: WiFi, Гал тогоо, Угаалгын өрөө, Зуух, Тагт

**Нөөцийн төлөв:**
- **Чөлөөтэй** — Захиалах боломжтой
- **Завгүй** — Хэрэглэж буй
- **Захиалсан** — Урьдчилан захиалсан
- **Засвартай** — Засварт байгаа, захиалах боломжгүй

### 7.3 Ажилтан оноох

Үйлчилгээнд ажилтан оноохдоо:

1. **"Ажилтан"** хэсэгт орно.
2. Ажилтнуудын жагсаалт, мэргэжил, ажлын хуваарь харагдана.
3. Үйлчилгээ бүрт хариуцах ажилтныг сонгоно.

### 7.4 Цаг захиалга удирдах

Цаг захиалгыг удирдахдаа:

1. **"Календарь"** хэсэгт орно.
2. Өдрөөр, долоо хоногоор, эсвэл сараар захиалгыг харна.
3. Захиалга дээр дарж дэлгэрэнгүй харна:
   - Харилцагчийн мэдээлэл
   - Захиалсан үйлчилгээ
   - Нөөц (ширээ, өрөө гэх мэт)
   - Оноосон ажилтан
   - Эхлэх, дуусах цаг

**Чатботоор цаг захиалга:**

AI чатбот нь хэрэглэгчдэд цаг захиалах боломж олгоно. Жишээ яриа:

> Хэрэглэгч: "Маникюр хийлгэх цаг бий юу?"
> Чатбот: "Маникюрын үйлчилгээ үнэ 25,000₮, 45 мин. Дараах цагуудад боломжтой: 10:00, 14:00, 16:00. Аль цагт захиалах вэ?"

## 8. Аналитик

### 8.1 Хянах самбар (Dashboard)

Нүүр хуудаснаас бизнесийн үндсэн мэдээллийг харна:

- Шинэ захиалга
- Шинэ мессеж
- Уншаагүй мэдэгдэл
- Өнөөдрийн борлуулалт

### 8.2 Тайлан хуудас

**Тайлан** хуудсанд дэлгэрэнгүй аналитик мэдээлэл харагдана.

**Хугацааны сонголт:**
- Сүүлийн 7 хоног
- Сүүлийн 30 хоног
- Сүүлийн 90 хоног
- Сүүлийн жил

#### Борлуулалтын тайлан

Дэлгэцийн дээд хэсэгт дөрвөн үндсэн тоон үзүүлэлт:

| Үзүүлэлт | Тайлбар |
|-----------|---------|
| **Нийт орлого** | Сонгосон хугацааны нийт борлуулалт (өмнөх үетэй харьцуулсан хувь) |
| **Захиалга** | Захиалгын тоо, дундаж захиалгын дүн |
| **Шинэ харилцагч** | Шинэ бүртгэгдсэн харилцагчийн тоо |
| **AI мессеж** | AI ашигласан мессежийн тоо / лимит |

**Графикууд:**
- **Орлого** — Хугацааны орлогын шугаман график
- **Захиалга** — Захиалгын тооны баганан график
- **Захиалгын статус** — Бялуу диаграм (хүлээгдэж буй, баталгаажсан, хүргэсэн гэх мэт)
- **Шилдэг бүтээгдэхүүн** — Хамгийн их борлуулалттай 5 бүтээгдэхүүн
- **Мессежийн статистик** — Харилцагч, AI, Админ мессежийн харьцуулалт

#### Хүргэлтийн тайлан

**Тайлан > Хүргэлт** хуудсанд:
- Нийт хүргэлтийн тоо
- Амжилттай хүргэсэн хувь
- Дундаж хүргэлтийн хугацаа
- Жолооч бүрийн гүйцэтгэл

#### AI Chatbot статистик

- **Нийт мессеж** — Сонгосон хугацааны нийт мессеж
- **AI хариулт** — AI-гийн хариулсан мессежийн тоо
- **Харилцагч мессеж** — Харилцагчдын илгээсэн мессеж
- **Хариулт хувь** — AI хариулсан хувь

#### Экспорт

Тайлангийн өгөгдлийг **Excel** эсвэл **CSV** форматаар татаж авах боломжтой.

### 8.3 AI зөвлөмж (AI Insights)

Тайлан хуудасны дээд хэсэгт **"AI Дүгнэлт"** карт байна. AI таны бизнесийн өгөгдлийг шинжилж, дараах зүйлсийг зөвлөнө:

- Борлуулалтын чиг хандлага
- Шилдэг бүтээгдэхүүний дүн шинжилгээ
- Харилцагчийн идэвхжилтийн үнэлгээ
- Анхаарах ёстой зүйлс (жишээ: хүлээгдэж буй захиалга их, цуцлалт өссөн гэх мэт)

AI дүгнэлт гурван төрлийн байна:
- **Эерэг** (ногоон) — Бизнес сайн явж байгаа тухай
- **Мэдээллийн** (цэнхэр) — Ерөнхий мэдээлэл, зөвлөмж
- **Анхааруулга** (шар) — Анхаарах шаардлагатай зүйлс

### 8.4 Өдөр бүрийн тайлан

Систем өдөр бүр и-мэйлээр тайлан илгээнэ:
- Өнөөдрийн захиалгын тоо
- Өнөөдрийн орлого
- Шинэ харилцагчийн тоо
- Мессежийн тоо
- Шилдэг бүтээгдэхүүн

## 9. Тохиргоо

### 9.1 Дэлгүүрийн мэдээлэл

**Тохиргоо > Дэлгүүрийн мэдээлэл** хэсэгт:

- **Нэр** — Дэлгүүрийн нэр
- **Лого** — Лого зураг
- **Тайлбар** — Товч танилцуулга
- **Холбоо барих утас** — Утасны дугаар
- **Хаяг** — Байршлын хаяг
- **Ажлын цаг** — Ажлын эхлэх, дуусах цаг

### 9.2 Баг удирдах

Олон хүн хамтран ажиллах боломж.

**Баг гишүүн нэмэхдээ:**

1. **Тохиргоо > Баг** хэсэгт орно.
2. **"Гишүүн урих"** товчийг дарна.
3. И-мэйл хаяг оруулна.
4. Эрх сонгоно:

| Эрх | Тайлбар |
|-----|---------|
| **Эзэн (Owner)** | Бүрэн эрх, дэлгүүр устгах боломжтой |
| **Админ (Admin)** | Бүх тохиргоо хийх боломжтой |
| **Ажилтан (Staff)** | Захиалга, чат харах, бүтээгдэхүүн засах |

5. Урилга илгээгдэнэ.

**Гишүүн устгах:**

Гишүүний хажууд байрлах **"Устгах"** товчийг дарна.

### 9.3 Мэдэгдэл

**Тохиргоо > Мэдэгдэл** хэсэгт мэдэгдлийн тохиргоог удирдана.

#### Мэдэгдлийн төрөл

Дөрвөн суваг байна:

| Суваг | Тайлбар |
|-------|---------|
| **Апп дотор** | Хянах самбар дээр хонхоор мэдэгдэнэ (үргэлж идэвхтэй) |
| **И-мэйл** | И-мэйл хаяг руу мэдэгдэнэ |
| **Push** | Вэб хөтчийн push мэдэгдэл |
| **Webhook** | Гадны системд JSON өгөгдөл дамжуулна |

#### Мэдэгдэл авах үйл явдал

Үйл явдал тус бүрт аль сувгаар мэдэгдэл авахыг тохируулна:

| Үйл явдал | Тайлбар |
|-----------|---------|
| **Шинэ захиалга** | Шинэ захиалга ирэхэд |
| **Шинэ мессеж** | Хэрэглэгчээс мессеж ирэхэд |
| **Нөөц дуусаж байна** | Бүтээгдэхүүний нөөц бага болоход |
| **Захиалгын төлөв өөрчлөгдөх** | Захиалгын төлөв шинэчлэгдэхэд |
| **Шинэ харилцагч** | Шинэ хэрэглэгч бүртгэгдэхэд |
| **Шилжүүлэг** | Чатботоос хүний оператор руу шилжүүлэхэд |

#### Мэдэгдлийн дуу

Апп дотор мэдэгдлийн дуу тохируулах боломжтой. Шинэ мэдэгдэл ирэхэд дуугарна.

#### Push мэдэгдэл тохируулах

1. **Тохиргоо > Мэдэгдэл** хэсэгт **"Push мэдэгдэл идэвхжүүлэх"** товчийг дарна.
2. Хөтөч зөвшөөрөл асуух тул зөвшөөрнө.
3. Push мэдэгдэл идэвхжсэний дараа хөтөч хаасан ч мэдэгдэл авна.

### 9.4 Webhook & API

Гадны системтэй холбоход ашиглана.

**Webhook URL:**

1. **Тохиргоо > Webhook & API** хэсэгт орно.
2. Өөрийн системийн webhook URL-ийг оруулна.
3. Ямар үйл явдлын мэдэгдэл авахыг сонгоно.

Webhook нь HMAC-SHA256 гарын үсэгтэй, 3 удаа дахин оролдох (exponential backoff) боломжтой, найдвартай дамжуулалттай.

**API Key:**

1. **Тохиргоо > Холболт** хэсэгт API Key харагдана.
2. **"Харуулах"** товчоор API Key-гээ харна.
3. **"Хуулах"** товчоор хуулна.
4. **"Шинэчлэх"** товчоор шинэ API Key үүсгэнэ (хуучин key ажиллахаа болино).

API Key-г гадны системээс бүтээгдэхүүн оруулах, захиалга авах зэрэгт ашиглана.

### 9.5 Хэл солих (MN/EN)

Temuulel платформ Монгол болон Англи хэлийг дэмждэг.

- Хянах самбарын хэл нь үндсэн Монгол хэлтэй
- Чатботын хариултын хэлийг **Тохиргоо > Chatbot тохиргоо > Хэл** хэсгээс тохируулна:
  - **Монгол** — Зөвхөн монгол хэлээр хариулна
  - **English** — Зөвхөн англи хэлээр хариулна
  - **Хоёулаа** — Хэрэглэгчийн хэлээр хариулна

### 9.6 Төлбөрийн тохиргоо

**Тохиргоо > Төлбөр хүлээн авах** хэсэгт:

- **QPay** — QPay мерчант мэдээлэл
- **Social Pay** — Social Pay тохиргоо
- **Банкны данс** — Банкны данс мэдээлэл

### 9.7 Дэлгүүр устгах

**Тохиргоо** хуудасны доод хэсэгт **"Аюултай бүс"** байна.

- **"Дэлгүүр устгах"** товчийг дарна.
- Бүх өгөгдөл бүрмөсөн устах бөгөөд сэргээх боломжгүй гэдгийг анхааруулна.
- Баталгаажуулсны дараа дэлгүүр устгагдана.

## Түгээмэл асуулт

### Чатбот яаж ажилладаг вэ?

AI чатбот нь хэрэглэгчийн мессежийг шинжилж, зорилгыг тодорхойлно:

| Зорилго | Жишээ мессеж | Хариулт |
|---------|-------------|---------|
| Бүтээгдэхүүн хайх | "Гутал байгаа юу?", "Цамц үнэ" | Бүтээгдэхүүн жагсаалт, үнэ |
| Захиалга шалгах | "Миний захиалга хаана байна?" | Захиалгын төлөв |
| Мэндчилгээ | "Сайн байна уу" | Welcome мессеж |
| Баярлалаа | "Баярлалаа" | Баярлалаа хариулт |
| Гомдол | "Би гомдож байна" | Хүний оператор руу шилжүүлэх |
| Хэмжээ | "M хэмжээ тохирох уу?" | Хэмжээний зөвлөмж |
| Төлбөр | "Яаж төлөх вэ?" | Төлбөрийн арга |
| Хүргэлт | "Хэзээ хүргэх вэ?" | Хүргэлтийн мэдээлэл |

### Нэг удаад хэдэн бүтээгдэхүүн оруулж болох вэ?

Excel/CSV импортоор хэдэн ч бүтээгдэхүүн нэг дор оруулж болно. Гэхдээ төлбөрийн планаас хамааран бүтээгдэхүүний лимит байна.

### Мессежийн лимит дуусвал яах вэ?

Мессежийн лимит дуусахад AI чатбот автомат хариулт өгөхөө зогсоно. Та гараар хариулах боломжтой хэвээр байна. Лимитийг нэмэхдээ план шинэчилнэ үү.

### Нөөц дуусахад яадаг вэ?

Бүтээгдэхүүний нөөц тохируулсан босго хэмжээнд хүрэхэд дэлгүүрийн эзэнд **"Нөөц дуусаж байна"** мэдэгдэл автоматаар илгээнэ. Чатбот нөөцгүй бүтээгдэхүүнийг санал болгохгүй.

### Олон хүн нэг дэлгүүрийг удирдаж болох уу?

Тийм. **Тохиргоо > Баг** хэсгээс гишүүд нэмж, эрх тохируулна. Owner, Admin, Staff гэсэн гурван эрхийн түвшин байна.

### Webhook яаж ажилладаг вэ?

Тодорхой үйл явдал болоход (шинэ захиалга, мессеж гэх мэт) таны тохируулсан URL руу JSON өгөгдөл илгээгдэнэ. n8n, Zapier зэрэг автоматжуулалтын хэрэгсэлтэй холбоход тохиромжтой.
